### Счётчик
Создай класс `Counter` с атрибутом `value`, равным 0.

Добавь методы:
- `increment()` — увеличивает значение на 1;
- `reset()` — сбрасывает значение на 0;
- `show()` — выводит текущее значение.

Пример:
```python
c = Counter()
c.increment()
c.increment()
c.show()  # Значение: 2
c.reset()
c.show()  # Значение: 0
```
